## POC&EXP使用

### 参数说明

```shell
optional arguments:
  -h, --help            show this help message and exit
  -u URL, --url URL     input url
  -f FILE, --file FILE  input file path
```

### 单个扫描

```shell
python3  1.py  -u  http://ip
```

### 批量扫描

```shell
python3  1.py  -f url.txt
```
